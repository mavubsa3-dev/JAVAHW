# 계산기 과제

## 구성
- App : 옵션 선택,  입력값 및 연산 결과 전달.
- ArithmeticCalculator : 연산 수행. 람다 및 스트림 통해 입력한 정수값 보다 큰 연산 값 반환
- BigIndex : 람다식 구현하기 위한 함수형 인터페이스
- CalculatorRep : 연산 결과 저장 및 리스트 반환
- OperatorType : 연산 기호 저장하기 위한 열거형 타입

### 기능
1. 연산
    - 숫자 입력 -> "."이 들어있는지 확인해서 정수, 실수형 파악해서 박싱
    - enum 활용해 기호 저장 후 입력한 기호와 비교해서 해당 연산 수행하도록 구현

2. 조회
    - 게터를 활용해 리스트 받아서 값 출력

3. 첫 번째 값 제거
    - 리스트 받아와서 remove() 활용해 첫 번째 인덱스 값 제거

4. 큰 값 보기
    - 숫자 입력 받은 후 리스트의 값들과 비교해서 입력값보다 큰 값들 출력
    - 람다식
      - 함수형 인터페이스 구현 후, 입력값과 리스트 App으로부터 받아서 검사 후 큰 값을 새 리스트에 담아서 반환

    - 스트림
       - Predicate<T> 통해 .filter()로 입력값 보다 큰 연산 값들 리스트로 반환
